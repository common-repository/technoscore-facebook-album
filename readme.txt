== Description ==

Technoscore Facebook Album is  best plugin for WordPress to add Facebook Album to any post or page.

= More details are as follows:- =
* It adds a 'Add Facebook Album' tag in Post Content Editor of post/page to insert shortcode of facebook album in post content area to show on frontend of the particular post/page.
* It provides options to add custom css class to change the look and feel of facebook album in shortcode.
* It display album images in popup.

== Installation ==

1. Use WordPress' built-in installer
2. Download the ZIP file from WordPress website
3. Extract the technoscore-facebook-album directory to your computer
4. Upload the technoscore-facebook-album directory to the /wp-content/plugins/ directory
5. Activate the plugin in the Plugin dashboard
